Bangkok Post Group — 76th Anniversary Project
โปรเจกต์หน้าเว็บพิเศษฉลองครบรอบ 76 ปี ของ Bangkok Post Group ซึ่งรวบรวมข้อมูลเกี่ยวกับมูลนิธิบางกอกโพสต์ (Bangkok Post Foundation) กิจกรรมส่งเสริมสังคม และระบบสนับสนุนการบริจาค (Donation System)

🌟 ฟีเจอร์หลัก (Key Features)
Responsive Landing Page: ออกแบบให้รองรับการแสดงผลทุกหน้าจอ (Desktop, Tablet, Mobile)

Donation System: หน้าส่วนการบริจาคที่แบ่งเป็นหมวดหมู่ (Tabs) ได้แก่

Bank Transfer: แสดงรายละเอียดเลขบัญชีธนาคารพร้อมปุ่ม Copy เลขบัญชีอัตโนมัติ

TrueMoney: รองรับการชำระเงินผ่าน Mobile Wallet

Credit Card: ระบบรองรับการกรอกข้อมูลบัตรเครดิต

Interactive Components: * ระบบสลับ Tab สำหรับดูข้อมูลการบริจาคแต่ละประเภท

QR Code Generator (Mockup) สำหรับการสแกนจ่าย

Toast Notification แจ้งเตือนเมื่อคัดลอกเลขบัญชีสำเร็จ

📁 โครงสร้างไฟล์ (File Structure)
index.html: โครงสร้างหลักของหน้าเว็บและส่วนของ Section ต่างๆ

style.css: ไฟล์สไตล์หลักและการจัดการ Layout (Flexbox/Grid) และ Media Queries

donation_section.html/css/js: ส่วนประกอบเฉพาะ (Modular) สำหรับระบบการบริจาค เพื่อให้ง่ายต่อการจัดการและไม่กระทบส่วนอื่น

script.js: ฟังก์ชันการทำงานทั่วไป เช่น การเลื่อน (Scroll) และ Interaction ต่างๆ บนหน้าเว็บ

🛠 เทคโนโลยีที่ใช้ (Tech Stack)
HTML5: โครงสร้างเว็บที่เป็น Semantic HTML

CSS3: ใช้ Custom Properties (Variables) ในการจัดการ Theme สี และใช้ Media Queries สำหรับ Responsive Design

Vanilla JavaScript (ES6+): จัดการ Logic การสลับ Tab, การคัดลอกข้อมูลลง Clipboard และฟังก์ชัน UI ต่างๆ โดยไม่พึ่งพา Library ภายนอก

🚀 วิธีการใช้งาน (Getting Started)
Clone หรือดาวน์โหลดไฟล์ทั้งหมดลงเครื่อง

เปิดไฟล์ index.html ผ่าน Browser หรือใช้ Extension เช่น Live Server ใน VS Code เพื่อดูผลลัพธ์

ตรวจสอบการเชื่อมต่อไฟล์ CSS และ JS ใน index.html ว่าเส้นทางถูกต้อง
